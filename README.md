# Worth It? Website

## Project info

Landing site for Worth It? — Reflect on what matters.

## How to run locally

Requirements: Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

Then open the local URL shown in the terminal (e.g. http://localhost:8080/).

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build & deploy

```sh
npm run build
```

The built files are in `dist/`. Deploy that folder to any static host (Vercel, Netlify, GitHub Pages, etc.).
